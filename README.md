

# Find Similar

This is a small project to test some React libraries:

- react-redux
- react-thunk
- styled-components
- jest
- encyme
- propTypes
- ant design

# What it does

You can input a movie/band/show/book/author/game and it shows similar content using [TasteDice API](https://tastedive.com/read/api)
